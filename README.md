# ilgiova237's Ultimate Optimization Kit  
  
A carefully curated, constantly updated collection of **free and open-source tools** to clean, optimize, debloat, and secure Windows 10 and Windows 11. Whether you're a gamer looking for lower latency, a power user who values privacy, or an IT technician repairing systems - this kit has everything you need.  
  
---  
  
## Structure  
  
The tools are organized by **optimization category**, then by **file type** (BAT, EXE, REG, PS1, FOLDERS) so you can instantly find what you're looking for.  
  
| Category | Description | What's inside |  
| :--- | :--- | :--- |  
| **Feature Adder** | Add missing Windows features | Scripts that enable Group Policy Editor, Hyper-V, Windows Sandbox, etc. |  
| **General Optimizations** | System-wide performance, cleanup, and repair | Tweaking scripts, portable system utilities, registry tweaks, PowerShell cleaners |  
| **Network Optimizations** | Lower ping, reduce bufferbloat, optimize DNS | TCP/IP optimizers, DNS benchmarks, network reset tools, gaming VPN helpers |  
| **Telemetry and Privacy** | Block trackers, remove bloatware, lock down Windows | Telemetry blockers, AI-removal scripts, firewall frontends, privacy dashboards |  
  
---  
  
## Getting Started  
1. Download the whole repository or clone it with `git clone https://github.com/ilgiova237/ilgiova237-s-Ultimate-Optimization-Kit.git`  
2. Browse the folder that matches the optimization you want.  
3. Run the scripts and tools **as Administrator** - most of them require elevated rights.  
4. Reboot after applying major changes.

## Important Notes  
- Always create a System Restore point before running any optimization tool.  
- Some scripts and tools stop Windows services and modify the registry. Use them on a fresh Windows install for the best results.  
- If you're unsure about a tweak, look inside the script - most are well commented.  
- This kit is for Windows 10 and Windows 11 (builds 19045 and later).
